MuSu - Stands for Music Selection. It is a web app that suggests music based on your mood and genre preferences, using the Spotify API. It's designed to make music discovery more fun, easy, and practical.

Features:
Spotify login with secure authentication
Mood selection page (Happy, Sad, Chill, Energetic, Angry)
Genre selection page generated from chosen moods
Randomized song suggestion system
Plays a short snippet of the suggested song
“Get Another Song” button
“Start Over” button
“Back” and “Logout” options
Displays song name, artist, and album cover

Tech Stack:
Python (Flask) - back-end routing and logic
Spotipy - communication with Spotify API
Spotify Web API - authentication, artist search, track retrieval
HTML / CSS - interface and layout
Sessions - storing user mood and genre selections
GitHub - version control and collaboration

Running on: https://musu-ypu7.onrender.com/

Usage:
Navigate to https://musu-ypu7.onrender.com/ in your browser.
Log in with Spotify.
Select your preferred moods and genres.
Enjoy the randomly selected song!
